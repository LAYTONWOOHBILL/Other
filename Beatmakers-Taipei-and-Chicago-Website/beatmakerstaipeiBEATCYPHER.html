BeatmakersTaiepi Beat Cypher 大隊接力


每個月舉辦的 "Beat Cypher 大隊接力"，提供一個良善平台，讓參與者使用提供的sample盡情地創造發揮，且能以beat會友，創造更優質的電子臥室音樂人環境。

規則 / Rules
    * 隨意取用提供的sample但至少得用一個 / use at least one sample from what we provide
    * 歌曲速度/ BPM: 不限 / free
    * 歌曲長度/ beat length: 1-2 分鐘左右 / around 1-2 min(s)
    * 繳交日期/ deadline: 06/28 (Fri.)
    * 完成的beat請轉mp3，檔案的名稱請打上各位的 producer name please send us mp3 with your producer name as the file name.
    * 想要在beat tape被Tag的人請提供facebook ID / provide your facebook ID so we can tag you
    * 直接用email附件寄到: beatmakerstaipei@gmail.com / attach the mp3 directly to the mail and send it to: beatmakerstaipei@gmail.com
